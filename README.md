# Network_qt

Qt/C++ based detect network interfaces and it's details.


# Qmake build


		git clone https://github.com/aks-arise1600/Network_qt.git
		mkdir qbuild && cd qbuild
		qmake ..
		make
		
# Cmake build


		git clone https://github.com/aks-arise1600/Network_qt.git
		mkdir cbuild && cd cbuild
		cmake ..
		make
